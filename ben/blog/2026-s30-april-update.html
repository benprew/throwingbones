Shandalar 30 - April Update

Tags: s30 programming

I've been excited to see how much interest there's been in Shandalar 30th Anniversary edition, original release announcement.

Here's what's changed since the last update:

  • Engine Improvements: migrated to mage-go MTG rules engine, this includes support for over 80% of the cards in the Shandalar universe
  • Android Version: you have to sideload it. The overworld works, but the duel screen doesn't support touch gestures yet. Get it here.
  • Game Balance: expanded wiseman boons, excluded P9/special cards from starting decks and stores (occasionally you could buy a Sol Ring).
  • Audio: the game has sound! I used the original sound effects though I might have the wrong effects in some places
  • Card Power scoring: I trained an ML model (BERT) to estimate card power, which I can use when generating starting decks and reward cards. This also means as new cards are added it will be able to build crappy starting decks and scale rewards.

Next Steps:

  • Improve AI opponent
  • Increase game engine card coverage to 100%
  • Add dungeons/castles

If you want to follow along or contribute, the project is on GitHub. Bug reports, card implementations, and play testing are all welcome! Check out the contributing guide to get started. Feel free to ask questions on the github repo or on reddit.
