Shandalar: 30th Anniversary Edition - May Update

Tags: s30 programming

Back with another monthly update on Shandalar: 30th Anniversary Edition. First bug report! Feydan reported it wasn't clear in the UI when creatures were marked to attack. I added a cool little animation when a creature is selected to attack.

Thanks to everyone sending bug reports and feedback. Please keep it coming.

Lots of little improvements this month, it's starting to feel more like a game instead of a tech demo. Now when I go to test something, I find myself actually playing through the game more. New game + difficulty screens, the AI is improving and you can fight the wizards! There's still a lot of work to do, but it feels good.

Highlights

  • New game screens, with difficulty and color selection!
  • Saves are grouped by play-through instead of showing all auto saves
  • [beta] Wizard Castles are available! When you go to a castle it shows you the duel screen for the wizard. I'm still working on adding a dungeon to the castles.
  • Random enemies scale better - before any random enemy could show up, now the longer you play the harder the enemies get
  • Better Mulligan - uses London Mulligan rules (can always mulligan, discard 1 card for each mulligan)
  • AI is getting better, understands combat better now, still makes some obvious mistakes
