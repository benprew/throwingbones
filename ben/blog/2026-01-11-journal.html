New Year's Resolutions

I don't usually think too much about resolutions because I try not to wait to change habits. But I have a few things I want to do this year.

1. V1 of Shandalar 30

This is the big goal for the year. I will be really happy if I can get a functional game out by EOY. I've got a good start, but there's still so much to do. I'm using AI and it helps a lot, but it still takes time. Time to spec out how the features should work. Time to write up docs to explain to the AI how it works. And time to review/test the changes AI has made.

My book group is currently reading "Vibe Coding" so I'll try to pick up ideas to improve implementation speed.

2. Teach a class?

This is a maybe, I'll see how I feel after Recurse Center. Also, I'm not sure what teaching CS even looks like anymore. AI is so transformational it's hard to know how to teach programming now.

3. Lift and stretch regularly

Not too much in this one. I already lift and stretch occasionally, but I'd like to make it more of a habit, at least 2x/wk for lifting and 3x/wk for stretching.

4. See family and friends more

Depending on what I do after Recurse center I'd like to try and see people more. If I stop working it will make it easier to see my parents and grandparents.
